Welcome to Medication Extractor's documentation!
================================================

This project was created for JHU EN.705.603.81.FA22 Creating AI-Enabled Systems.

The project is based on a topic of your choice and required an end-to-end design that covers the 6Ds of creating AI-enabled systems.
The 6Ds include decomposition, domain expertise, data, design, diagnosis, and deployment.

This effort extracts medication names from text and maps them to their RXNorm CUI codes.
